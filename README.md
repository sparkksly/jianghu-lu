# 江湖路 · Jianghu Lu

> 一个武侠卡牌 roguelike：**排招 → 观战**的时间轴对决。从小渔村打到华山论剑。

## ▶ 在线试玩

### **https://sparkksly.github.io/jianghu-lu/**

浏览器直接打开即可（无需安装、无需下载 Godot）。首次加载约几秒。

## 玩法一览

- **排招**：把招式拖到 **12 拍**的时间轴上，安排出手时机；前摇/命中/后摇都占拍。
- **观战**：双方计划同时结算；底部时间轴有游标从左扫到右，两条泳道标出每拍的关键事件（大招 / 打断 / 格挡 / 闪避…）。
- **气（体力）跨回合管理**：不满回，爆发后要喘息——内功续航 vs 外功爆发。
- **连招**：把招式按配方相邻排好可融合，如 三连腿法 → 连环踢，攻+防+投 → 乾坤大挪移。
- **roguelike**：一局是一段江湖征程，逐场变强的对手，血量跨战延续。
