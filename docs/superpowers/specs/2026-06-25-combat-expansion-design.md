## 一、核心新轴：距离（本次最大的新机制）

在"时间轴"之上叠一层"空间"博弈。

- **一条共享距离** `D ∈ {贴身=0, 中=1, 远=2}`，**开局中距(1)**，先抢距离。
- **步法改距离（同时结算→博弈）**：
  - **上步(进)**：`D−1`（拉近）
  - **撤步(退)**：`D+1`（拉开）+ 回气 / 抢下回合先手
  - **绕步**：不改 D，换先手 / 绕开正面
  - 步法也是占拍的招式；**双方同拍一进一退 → 抵消**（你进身他正好后撤，就贴不上）。
- **招式有"适用距离"，在命中那一拍检查当前 D**：
  - 对不上 = **够不着(落空)**，无伤且露破绽（类似挥空）。
- **击退**词缀：命中把对手 `D+1`（推开）。部分攻击命中**拉近** `D−1`（上步攻击）。
- **可读性**：观战时间轴把"距离变化"做成事件标记；排招时每张牌标其适用距离。

## 二、技法家族（组织招式的轴 = 门派专精的轴）

原则：**每个动作都具体、有画面感**（不要"轻踢/重踢"这种数值标签），并落到帧博弈 + 距离上。下面是**起手词汇草案**（帧/距离/词缀为示意，实现时调）。

**通用基础（人人可用，弱而可靠）**：一个直拳 + 扫堂腿 + 格挡 + 闪身 + 撤步 + 擒拿。任何门派开局都能用。

### 拳法（适用距离：贴身）
- **直拳**：2拍、快、**打断**、抢拍——笔直试探拳
- **摆拳**：3拍、中伤、绕侧——抡圆的勾击
- **崩拳**：4拍、**重击**、直进——形意崩拳，沉重一击
- **贯耳**：双拳、2段

### 掌法（适用距离：贴身~中；内力载体）
- **推掌**：**击退**(D+1) + 可挂自身**气劲/护体 buff**
- **下劈掌**：过顶重劈、**破防/破霸体**
- **拍掌**：快、轻
- **双推掌**：2段

### 指法（适用距离：贴身~中；部分远程指劲）
- **戳指/点穴**：精准、命中→**定身/点穴**(跳过1-2拍)
- **弹指**：**远程**小伤(指劲)

### 腿法（适用距离：贴身~中；现有，重做为有画面感）
- **弹腿**：快直踢（替代"轻踢"）
- **扫堂腿**：横扫下盘、**绊势/破架**（替代"扫腿"）
- **侧踹**：重、**击退**(D+1)（替代"重踢"）
- **鞭腿**：旋身、中伤、广

### 身法/步法（无适用距离；改距离 / 防御 / 节奏）
- **闪身**：让一拍来招**挥空**（现有 dodge）
- **上步**：`D−1`，贴身
- **撤步**：`D+1` + 回气 + 抢先手
- **绕步**：换先手 / 绕正面
- **垫步**：诱敌挥空

### 擒拿（适用距离：贴身）
- **擒拿**：破格挡 / 投技

## 三、武器系统（捡装备 → 基础攻击整组换）

武器是 roguelike 道具：**装备后，你的基础攻击家族整组换成兵器版**（同一组"招式槽"换皮换数值，手感连续），并带兵器被动与 reach。

- **剑**（适用距离 贴身~中）：刺 / 撩 / 挑 / 劈 / 格剑 —— 快、精准、**破防**、料敌(打断)
- **刀**（适用距离 贴身~中）：劈 / 砍 / 缠 / 扫刀 —— 重、霸道、**流血**
- **棍·枪**（适用距离 **中**）：扎 / 扫 / 挑 / 抡 —— **长(中距压制，徒手难近身)**
- 例：直拳→直刺，推掌→撩剑。门派分"徒手派/剑派/刀派…"。

---

## 四、暗器（消耗品）

- **远程(中~远)、有数量、用一个少一个**；与"气"并列的另一种资源，roguelike 里补货。
- **飞镖**(必中小伤) / **毒针**(施中毒) / **袖箭**(连发)。逼对手"要么贴上来要么吃毒"。

## 五、状态系统（StatusEffect，通用持续 N 拍/回合）

- **中毒**：DoT，每拍/回合掉血，**可叠层**；来源暗器/毒掌/毒功门派。
- **流血**：行动时额外掉血；来源刀/利器。
- **内伤**：降目标 **回血/回气/防御** 若干回合，需疗伤解；来源重击/"七伤式"自损(也挂自己)。
- **控制**：定身/点穴(跳过若干拍)；来源指法。
- **增益(buff)**：
  - **金钟罩/铁布衫**：限时 +防御
  - **气劲**：限时 +攻击/+伤害
  - **护体真气**：限时霸体
  - **龟息**：+防御 −气(取舍)
  - **回春**：每回合回血
- 进入/离开时改属性；每拍结算 DoT/回血。映射到事件→观战标记 + 浮字(治疗绿/伤害红)。

---

## 六、攻防属性

- combatant 级 **攻击力 `atk` / 防御力 `def`**。
- 伤害 = `招式基础 × f(atk) − g(def)`（公式实现时定）。
- buff 改 atk/def（金钟罩↑def、气劲↑atk）；内伤↓def。
- 这是状态系统的承载：很多 buff/debuff 就是临时改 atk/def。

## 七、门派（混合制）

- **开局选一派**：定**起手招**(专精某技法家族/兵器) + 一个**标志机制**(一种状态/资源)。
- **途中跨派学**：遇大师/得秘籍可学别派招；**正邪/门派门槛**影响能学什么（接 roguelike run 结构、善恶侠名）。
- **起手门派候选**（原型→机制/状态→占位名，**用原创名避开金庸 IP**）：
  - 刚猛外功 → **内伤**（铁掌门）
  - 毒功暗器 → **中毒**（五毒教）
  - 柔劲防反 → **防御 buff / 借力打力**（玄门太极）
  - 轻身内功 → **续航 / 闪避抢拍**（云踪派）
  - （刀法**流血** 可作第 5 派后补）

## 九、已确认的设计决策

- 职业 = **混合制门派**：开局定底 + 途中跨派学。
- 招式组织轴 = **技法家族**（拳/腿/掌/指/暗器/兵器/身法），动作要**具体有画面感**。
- **加距离轴**：共享三档(贴身/中/远)，开局中距，步法改距离(同时结算)，命中拍查距离。
- **武器**整组替换基础攻击 + 被动；**暗器**消耗品远程施毒。
- 状态：中毒/流血/内伤/控制/增益；**攻防属性**承载 buff/debuff。
- IP：金庸特定名(降龙/逍遥/北冥等)**只作灵感**，落地用原创/通用名。
- 第一个 spec = **切片 1**（待本愿景文档审定后再单独写）。
