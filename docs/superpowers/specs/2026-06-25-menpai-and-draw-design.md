## 一、两门派(刚柔对照)

| | **少林** | **武当** |
|--|--|--|
| 气质 | 刚 / 硬 / 进 | 柔 / 化 / 反 |
| 距离 | **贴身硬刚**,不退(棍补中距) | **中距游走**,借力 |
| 核心 | 重击破防 + **护体扛揍** + 内伤 | **防反借力** + 轻身抢拍 |
| 标志状态 | **内伤**(debuff)、**护体**(减伤 buff) | **借力反击**(防守成功窗口) |

> IP:用真实/通用名(罗汉拳、韦陀掌、易筋经、金钟罩、太极拳、绵掌等);避开金庸特定名(纯阳无极功→可改"两仪心法"等)。

### 少林 · 招式池(抽卡进攻池起始内容)
- **罗汉拳系**:直拳 / 摆拳 / 崩拳(刚猛拳,贴身)
- **韦陀掌**:重掌,贴身~中,破霸体
- **大力金刚指**:破防戳,贴身~中
- **龙爪手**:刚猛擒抓,贴身
- **少林棍**:中距长兵 扫/劈(给少林一个中距手段)

### 武当 · 招式池
- **太极拳(化劲)**:防反核心
- **绵掌**:柔掌,中距,后发
- **武当长拳**:连招拳
- **太极剑**(可选/后补):剑·精准
- *(梯云纵轻功 → 作为武当风味的身法,先用通用身法代;后补)*

### 4 个基础连招(配方 → 融合结果 + 效果/状态)
**少林**
1. **罗汉连拳**　直拳→摆拳→崩拳(拳法×3)→ 刚猛三连,**末段命中挂【内伤】**。
2. **金刚伏魔**　格挡→韦陀掌(重掌)→ 重击+破霸体,**给自己挂【护体】**(几拍受伤减免)。

**武当**
3. **四两拨千斤**　格挡/闪身(化劲成功)→ 绵掌(反击)→ **防御/闪避成功后那一掌伤害大增**(借力反击窗口)。
4. **太极云手**　绵掌→绵掌(柔掌×2)→ 连绵化劲,**命中改距离±1(走位)+ 抢下回合先手**。

> 连招沿用现有「相邻招式匹配配方 → 融合压缩」机制;新增配方(拳法×3、防+重掌、化劲+反击掌、柔掌×2)。「化劲成功」需防守成功的判定(见状态三)。

## 二、最小状态系统(只做这 3 个;中毒/流血留给毒派)

通用 `StatusEffect`(持续 N 拍/回合,挂在 combatant 上):
- **内伤(debuff)**：降目标 **回血/回气 与（暂用）受击韧性** 若干回合;来源 罗汉连拳/重击。(完整「降防御」等攻防属性在攻防属性切片做;本切片先降回气/回血。)
- **护体(buff)**：自身 **受伤减免 X%** 若干拍;来源 金刚伏魔。
- **借力反击(窗口)**：**格挡/闪身成功的那一拍后开一个短窗口,期间你的下一记攻击伤害 +大幅**;来源 四两拨千斤(及武当被动)。

事件→观战标记/浮字:内伤(紫/灰)、护体(金盾)、借力(亮)。映射进 `CombatFeed`。

## 三、抽卡手牌系统

- **固定手牌(必在 · 不消耗 · 可无限排)**：上步 / 撤步 / 格挡 / 闪身 / 擒拿。防守与走位永远可靠。
- **进攻牌**：每回合从你的**进攻池**「**有放回**」随机抽 **6 张**(**可能重复**,如两张直拳)。**每张一次性——排一次消耗**(从 timeline 移除则退回手牌)。下回合重抽。
- **连招靠抽**:罗汉连拳(拳×3)要这回合手里**真有 3 张拳**才拼得出。
- **构筑深度**:进攻池越精简(集中拳法),越稳定抽够连招组件;越杂越"手风不顺"。**门派定起始进攻池**;途中学招扩池(接 roguelike 奖励,后续切片)。
- **排招交互改动**:进攻从「调色板可无限拖」变为「**手牌,排一次消耗**」;工具牌仍可无限。这是 plan_phase 的一次模型改动。

## 六、已确认决策
- 两门派 = **少林(刚/贴身/护体·内伤)** vs **武当(柔/中距/借力反击)**;刚柔对照。
- 招式:少林棍换下般若掌(精简掌法、补中距);武当用太极拳/绵掌/武当长拳(+太极剑可选)。
- 4 连招 + 3 最小状态(内伤/护体/借力)如上。
- 抽卡:**有放回抽6进攻(可重复)+ 一次性消耗;固定工具牌无限;每回合独立重抽;门派定起始池**。
- IP:用通用/原创名,避开金庸特定名。

## 七点五、修订(用户 2026-06-25)
- **先做抽卡(子切片 A)**;**B、C 合并**(A 之后一起做)。新顺序:**A → (B+C)**。
- **状态精简为 2 个**:
  - **护体** = **基础防御力上涨**(一个简单的防御值,减伤;= 攻防属性里"防御"的轻量版,本切片随护体一起引入最小防御值)。
  - **借力** = **成功格挡/闪避后,下一记攻击百分比增伤**(简单窗口)。
  - **内伤暂不做**(以后毒派/攻防属性再说)。
- 因此 **罗汉连拳** 效果改为「刚猛三连高伤 / 破防」(去掉内伤);**金刚伏魔** 给【护体(+防御)】;**四两拨千斤** = 借力(防御/闪避成功后增伤);**太极云手** 不变。
