# 武侠招式资料汇总（为头脑风暴准备）

> 收集自金庸、古龙等经典武侠小说，以及《金庸群侠传》《太吾绘卷》《活侠传》等武侠游戏。
> 仅限武侠题材。下面已按「招式类型」分类，并初步标注了可映射到本游戏 `Move` 机制的设计点
> （前摇/命中/后摇、伤害、体力、tag、打断/霸体/重击，以及拟新增的状态）。

## 一、金庸体系

### 掌法（重攻击 / 连招核心）
- **降龙十八掌**（萧峰、洪七公、郭靖）— 至刚至猛。十八式招名取自《易经》：
  亢龙有悔、飞龙在天、见龙在田、鸿渐于陆、潜龙勿用、突如其来、利涉大川、
  震惊百里、或跃在渊、双龙取水、鱼跃于渊、时乘六龙、密云不雨、损则有孚、
  龙战于野、履霜冰至、羝羊触蕃、神龙摆尾。
  - 设计点：天然是一套「连招」(combo) —— 多式按顺序排进时间轴可触发增伤/霸体。
    「亢龙有悔」可做高伤重击（is_heavy + super_armor），「神龙摆尾」做后撤反击。
- **黯然销魂掌**（杨过）— 情绪驱动，威力随状态变化 → 可做「血量越低伤害越高」的特殊缩放。
- **七伤拳**（谢逊）— 「先伤己再伤人」→ 高伤但对自己造成内伤(self-debuff)，呼应反噬机制。
- **玄冥神掌**（阿紫被掌伤）— 寒毒 → 命中施加「中毒/寒毒」DoT。
- **太极拳 / 太极剑**（张三丰）— 以柔克刚、借力打力 → 做成「格挡后反击」或「化解」型防御招。
- **龙象般若功**（金轮法王）— 层数越高威力指数增长 → 「蓄力层数」机制。
- **天山六阳掌 / 铁掌**（裘千仞）— 标准刚猛掌法。

### 剑法 / 刀法
- **独孤九剑**（令狐冲）— 「无招胜有招」「料敌先机」→ 做成「打断/破招」专精（can_interrupt）。
- **六脉神剑**（段誉）— 指力化剑气，远程 → 远程攻击 tick。
- **辟邪剑法 / 葵花宝典**（自宫武功）— 极快但有代价 → 极低前摇高速，但永久削弱某属性。
- **胡家刀法 / 苗家剑法**（飞狐外传）。

### 指法 / 暗器
- **一阳指 / 六脉神剑**（段氏）— 点穴 → 命中施加「定身/眩晕」(stun) 短控。
- **弹指神通**（黄药师）— 暗器/指劲弹射。
- **满天花雨**（暗器流）— 多段命中（hit_offsets 多个）。

### 内功（被动 / 增益资源）
- **易筋经、洗髓经、神照经**（疗伤/续航）→ 回血/解内伤 buff。
- **九阳神功**（自动疗伤、内力深厚）→ 每回合回体力/回血的持续 buff。
- **九阴真经**（攻防兼备）。
- **小无相功**（以内力催动他人招式）、**紫霞神功**、**北冥神功 / 化功大法 / 吸星大法**（吸取对手内力）
  → 「吸体力/吸 buff」机制：命中时偷取对手体力。

### 轻功（位移 / 闪避 / 抢拍）
- **凌波微步**（段誉）— 按八卦走位，可闪避 → 强化版 dodge，闪避并位移抢拍。
- **梯云纵**（武当）、**神行百变**、**飞燕七式** → 提升先手/降低被命中。

## 二、古龙体系（偏「一招制敌」、重意境）
- **小李飞刀**（李寻欢）— 「例不虚发，一刀封喉」→ 必中高伤暗器，但前摇长/冷却长（一回合一发）。
- **灵犀一指**（陆小凤）— 两指夹住任何兵器/暗器 → 攻防一体，可「接招/弹反」(parry)。
- **天外飞仙**（叶孤城）— 居高下击、剑气彻骨 → 终结技式高伤剑招。
- **嫁衣神功** — 「需毁掉自身再重练」→ 高风险反噬流，呼应七伤拳/走火入魔。
- **明玉功** — 永续吸取对方内力 → 持续吸体力 buff。
- **回风舞柳剑、神剑诀、飞燕七式** — 飘逸剑法/轻功。
- 兵器谱排名（天机棒 > 子母龙凤环 > 小李飞刀）可做「武器/招式稀有度」分级灵感。

## 三、武侠游戏里的「状态 / 属性」机制（直接可借鉴）

### 负面状态（debuff）
- **中毒**：多种毒，每回合掉血；毒值 > 抗毒则致命（《太吾绘卷》）→ DoT，可叠层。
- **流血**：行动时额外掉血（攻击/移动触发）。
- **内伤 / 伤势**：分轻重，降低主属性与防御，需更高级药/疗伤招才能解。
- **走火入魔 / 内息紊乱**：修炼反噬，攻防下降、出招出错 → 可作高风险招式的反噬代价。
- **定身 / 眩晕 / 点穴**：短控，跳过若干拍。
- **破防 / 卸力**：降低目标防御若干回合。

### 正面状态（buff）
- **金钟罩 / 铁布衫**：持续数回合的高防御（刀剑只留白痕）→ 防御 buff，可设「消耗体力维持」。
- **龟息功**：+内力/+防御，但 -体力 → 防御换续航的取舍。
- **霸体 / 护体真气**：免疫打断（已有 super_armor，可做成限时 buff）。
- **气劲提升 / 内力护体**：限时 +攻击力 / +伤害减免。
- **回春 / 疗伤**：每回合回血。

## 四、映射到本游戏现有机制的初步想法
现有 `Move`：`kind(攻/防/闪/拿)`、`startup/active/recovery`、`damage`、`stamina_cost`、
`tags`、`can_interrupt(打断)`、`super_armor(霸体)`、`is_heavy(重击)`、`hit_offsets(多段)`。

建议明天讨论新增的字段 / 系统：
1. **攻击力 / 防御力属性**（combatant 级）：伤害 = 招式基础 × 攻击系数 −（目标防御）。
   金钟罩 = 限时 +防御 buff；气劲 = 限时 +攻击 buff。
2. **状态系统 StatusEffect**（持续 N 拍/回合）：中毒、流血、内伤、眩晕、+防御、+攻击、回血、吸体力。
   - 每拍结算 DoT/回血；进入/离开时改属性。
3. **连招进阶**：把「降龙十八掌」「独孤九剑」做成有顺序加成的招式族（已有 combo 框架）。
4. **反噬流**：七伤拳/嫁衣神功 → 高伤但给自己挂内伤/走火入魔。
5. **控制 & 破招**：一阳指→眩晕；独孤九剑→打断专精；灵犀一指→弹反。
